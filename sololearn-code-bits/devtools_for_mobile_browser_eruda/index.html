Mobile browser DevTools: Eruda

Browse the demo on your phone: https://eruda.liriliri.io/

In order to try it for your or for different sites, execute the script below on browser address bar. Another option is to add bookmark for this code and open the bookmark on testing site, it works [1]:

javascript:(function () { var script = document.createElement('script'); script.src="//cdn.jsdelivr.net/npm/eruda"; document.body.appendChild(script); script.onload = function () { eruda.init() } })();

  • Home page eruda.liriliri.io with DevTools Demo
  • GitHub https://github.com/liriliri/eruda
  • Documentation README.md

Features

  • Console: Display JavaScript logs.
  • Elements: Check DOM state.
  • Network: Show requests status.
  • Resource: Show localStorage, cookie information.
  • Info: Show URL, User Agent info.
  • Snippets: Include snippets used most often.
  • Sources: HTML, JS, CSS source viewer.
